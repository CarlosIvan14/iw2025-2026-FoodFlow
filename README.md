# FoodFlow · POS Vaadin (Spring Boot)

Aplicación POS para restaurantes (mesas, pedidos, cocina, caja, inventario y analítica) construida con **Spring Boot 3** + **Vaadin Flow 24**.

## Requisitos

- **Java 21 (LTS)** instalado y `JAVA_HOME` apuntando a esa versión  
  - Verifica con: `java -version`
- **Maven Wrapper** incluido en el repo (`./mvnw` / `mvnw.cmd`) – no necesitas Maven global
- (Dev) Vaadin en modo desarrollo puede descargar Node automáticamente; no es obligatorio tener Node instalado.

Linux/Mac:
